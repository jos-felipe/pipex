<h1>Description</h1>
<p>
This project is about hanling pipes.

Your program will be executed as follows:
```bash 
/pipex file1 cmd1 cmd2 file2
```

It must behave exaclty the same as the shell command below:
```bash 
$> < file1 cmd1 | cmd2 > file2
```
</p>
